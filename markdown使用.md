# Markdown 基本使用方法
## 基本语法
### 1. 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```
### 2. 字体
```markdown
*斜体文本* 
**粗体文本** 
***粗斜体文本***
~~删除线~~
```
### 3. 列表
#### 无序列表
使用星号(*)
```markdown
* 第一项
* 第二项
* 第三项
```
#### 有序列表
使用数字并加上 . 号来表示：
```markdown
1. 第一项
2. 第二项
3. 第三项
```
#### 列表嵌套
列表嵌套tab
```markdown
1. 第一项：
   * 第一项嵌套的第一个元素
   * 第一项嵌套的第二个元素
2. 第二项：
   * 第二项嵌套的第一个元素
   * 第二项嵌套的第二个元素
```
### 4. 链接
链接使用方法如下：
```markdown
[链接名称](链接地址)
```
### 5. 图片
图片语法格式如下：

```markdown
![alt 属性文本](图片地址 "可选标题")
```
## 高级功能
### 表格
```markdown
| 表头 | 表头 |
| ---- | ---- |
| 单元格 | 单元格 |
| 单元格 | 单元格 |
```
对齐方式：
```markdown
| 左对齐 | 右对齐 | 居中对齐 |
| :--- | ---: | :---: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
```
### 4. 分割线

```markdown
*****************
```
